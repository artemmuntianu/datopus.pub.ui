Link BigQuery with Analytics

Proceed to the next step if you already have Google BigQuery and Google Analytics linked.

Now that you have both Analytics and BigQuery set up, let's link them together.

Link a Google Analytics 4 property to BigQuery

  1. In Admin, under Product Links, click BigQuery Links.

    • You must be an Editor or above at the property level to link an Analytics property to BigQuery.
    • You must also use an email address that has OWNER access to the BigQuery project.
  2. Click Link.
  3. Click Choose a BigQuery project to display a list of projects for which you have access.
  4. Select a project from the list, then click Confirm.
  5. Select a location for the data.
  6. Click Next.
  7. Select Configure data streams and events to select which data streams to include with the export.
  8. Click Done.
  9. Select Daily (once a day) export of data.
  10. Click Next.
  11. Review your settings, then click Submit.

When you start seeing data

Once the linkage is complete, data should start flowing to your BigQuery project within 24 hours.
One file will be exported each day that contains the previous day's data. Generally, during early afternoon in the time zone you set for reporting.
